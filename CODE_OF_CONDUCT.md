# Code of Conduct

이 프로젝트는 Contributor Covenant v2.1을 따릅니다.

전문은 https://www.contributor-covenant.org/version/2/1/code_of_conduct/ 에서 확인할 수 있습니다.

## 우리의 약속

우리는 나이, 외모, 장애 유무, 민족, 성별 정체성과 표현, 경험 수준, 국적, 인종, 종교, 성적 지향과 관계없이 모든 구성원에게 괴롭힘 없는 경험을 보장합니다.

## 표준

긍정적인 행동의 예:
- 다른 의견과 관점을 존중하는 태도
- 건설적인 비판을 우아하게 주고받기
- 커뮤니티의 학습과 성장에 기여

받아들일 수 없는 행동:
- 성적 표현이나 이미지의 사용
- 모욕적·경멸적 코멘트, 인신공격
- 공개·비공개적 괴롭힘
- 타인의 개인정보 무단 공개

## 집행

위반 신고는 `conduct@authlens.dev` (운영 시 실제 메일로 교체)로 보내주세요. 모든 신고는 비공개로 검토됩니다.

## 라이선스

이 행동 강령은 Contributor Covenant v2.1, CC BY 4.0 하에 배포됩니다.
